# AI-feeded-translator-Chinese-English-
Quick project of a chrome extension that translates between English-Chinese feeded by AI (Deepseek)

Firstly, open .env file and add you API from deepseek
Then start your server on Python for the testing, it's needed to install the packages for the private server before running the code. 
Once the server is updated locate the route of the folder with the translator files, and once you have the route type -python app.py to start running the server

Check on IP of the server to make sure is working -- " http://127.0.0.1:5000/health " paste this on your web browser.

Finally add the extension on chrome extensions by selecting imbeded extension, and picking the translator

The add on works just like google translator with right click on the text that you have on your screen and selecting wheter if you want to translate to English or Chinese
